# Welcome to the iPhone 15 Pro Clone

This repository contains a clone of Apple's iPhone 15 Pro website, built using:

• [GSAP](https://gsap.com/)

• [React](https://react.dev/)

• [Three](https://threejs.org/)

• [Tailwind CSS](https://tailwindcss.com/)
